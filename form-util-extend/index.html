Form Extend

Doc: Forms

Methods

FormUtil._extendInput($ele)

封装Input, $ele为被封装的form元素, 封装后可调用如下方法:

Method Name example Description
disable $ele.disable() 将该元素disabled(不能填写)
enable $ele.enable() 将该元素enable(可填写)
getValue $ele.getValue() 得到该元素的值
clear $ele.clear() 将该元素的值清空

FormUtil._extendSelectPicker($ele)

封装selectPicker, $ele为被封装的form元素, 封装后可调用如下方法:

Method Name example Description
disable $ele.disable() 将该元素disabled(不能选择)
enable $ele.enable() 将该元素enable(可选择)
getValue $ele.getValue() 得到该元素的值
clear $ele.clear() 将该元素的选中项清空
clearOptions $ele.getSelectedItems() 得到选择的value,text的值对象, 例如{"value": "1", "text": "Option A"}

FormUtil._extendSelectize($ele)

封装selectize, $ele为被封装的form元素, 封装后可调用如下方法:

Method Name example Description
disable $ele.disable() 将该元素disabled(不能选择)
enable $ele.enable() 将该元素enable(可选择)
getValue $ele.getValue() 得到该元素的值
clear $ele.clear() 清空选中项
clearOptions $ele.clearOptions() 清空所有selectize中的options
getSelectedItems $ele.getSelectedItems() 得到选中的对象
