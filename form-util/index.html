Methods

FormUtil.addHiddenFields($form, hiddenParams)

给指定form添加隐藏的input field(若该form中有同名input域，则更新input的值为最新的值)

Name Type Default Description
$form jquery object example: $("#form1")
hiddenParams array of hash [] example:
[{name: "courseId", value: 1}, {name: "courseNme", value: "math"}]

FormUtil.clearFields($form, [fieldNames])

删除指定form下的name为fieldNames的表单域，若fieldNames不填，则删除全部

Name Type Default Description
$form jquery object example: $("#form1")
fieldNames array of string 可不写 example: ['courseId','courseNme']

FormUtil.getFieldValues()

返回值: hash数组, example:{"courseId":1, "courseName": "math"}

得到表单域的值,有四种传参方式:

FormUtil.getFieldValues($form)
FormUtil.getFieldValues($form, fieldNames)
FormUtil.getFieldValues($form, getEmpty)
FormUtil.getFieldValues($form, fieldNames, getEmpty)
Name Type Default Description
$form jquery object example: $("#form1")
fieldNames array of string 可不写 只得到指定name的值的域
example:['courseId','courseNme']
getEmpty boolean true getEmpty为false将不得到任何表单域的值
